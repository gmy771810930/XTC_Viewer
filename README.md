# XTC_Viewer
XTC文件预览器

需要Python环境，直接运行即可。

默认以文件原分辨率显示，可以选480×800，960×1600，528×792，1056×1584预览效果

代码由Deepseek生成
ai提示词：
1.使用Python写一个图形化程序，用于预览XTC或XTCH格式
2.在程序目录下log文件夹（没有则自动创建）下输出所有日志并生成 日期时间.log 文件以便排障
3.自动下载安装缺失的依赖
4.程序默认全分辨率查看，可以额外选择X4（480×800），X4双倍分辨率（960×1600），X3 （528×792），X3双倍分辨率（1056×1584）模式预览
